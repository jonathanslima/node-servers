## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

* NodeJS is required.

### Installing

1. Go into any of the following folders 
	* */express*
	* */koa*
	* */ndodejs*
	* */restify*
2. run: *npm i*
3. run: *node server.js*
